# Улучшение материала

Вы можете предложить любые свои правки в материал:

- Исправления опечаток
- Указание на интересные материалы, из которых можно дополнить контент
- Дополнительные разъяснения существующего материала
- Новые яркие интересные примеры и объяснения
- Новые задачи или решения уже имеющихся
- Даже новые статьи!

Если ваши предложения привели к добавлению нового контента **(кроме сообщений об опечатках!)**, то мы можем добавить ваши данные (имя, фото, пару слов про вас и ссылки) на страницу авторов, если вы сами этого захотите.

Предложить свои правки можно двумя способами:

1. Напрямую отредактировать материал на GitHub и создать Pull Request
2. Связаться со мной через Телеграм ([@gwynerva](https://t.me/gwynerva)) или по почте ([newpetya@inbox.ru](mailto:newpetya@inbox.ru))

# Как использовать перекрестные ссылки

Перекерсные ссылки внутри материалов учебника создавать делать не надо, если элемент, на который создается ссылка, находится в том же разделе (под тем же H1, H2, H3 заголовком).

Если же уникальный элемент находится в другом разделе, то ссылаться на него в новом разделе можно не больше одного раза в каждой "области видимости".
Например, можно сослаться с ссылкой один раз в каком-нибудь параграфе этого раздела, и по одному разу в скрытых секциях блоков, например подсказках, решениях задач и так далее.

В задачах вообще желательно если ссылки нет прямо в условии, то дублировать ее по необходимости один раз в каждой секции, где она нужна (подсказки, решение, ответ, примечание).

# Использование "Внешних связей"

Во "внешние связи" стоит добавлять **только** ресурсы, которые реально способны дать дополнительную полезную информацию для читателя, который уже ознакомился с материалом на сайте.

Пример: имеем группу материалов "Уравнения" в учебнике "Основы Математики"

**Отличные кандидататы во внешние ресурсы:**

- Сборник текстовых задач, сводящихся к уравнениям.
- Книжки, целиком посвященные разным видам уравнений (например диофантовым, кубическим и так далее).
- Хороший и толковый школьный учебник соответствующего класса с большим количеством разных задач на уравнение.

Не обязательно, чтобы материал из ресурса как-то использовался в "Открытой Математике".
Можно просто указывать ссылки на реально полезные ресурсы, даже не заимствуя из них ничего в сам материал статей.

**Плохие кандидаты во внешние ресурсы:**

- Ссылка на пост из социальной сети, откуда была взята интересная задача. _Никакой полезной информации кроме уже перенесенной задачи там нет._
- Ссылка на биографический сайт, с которого была взята картинка для статьи. _Опять же, остальной материал этого сайта к статье отношения не имеет._

Ещё раз — если на внешнем ресурсе нет чего-то такого, что действительно дополняет уже написанный материал, добавлять такой ресурс **не надо!**

В случае, когда ссылку по закону **обязательно надо** добавить (когда это затребовал обладатель авторских прав), ее можно можно оформить прямо внутри текста статью с помощью тега `<Ref />`:

```md
На изображении ниже можно увидеть первые наброски Сатурна и спутников, которые тогда считались его "кольцом".
Это сканы из книги Гюгенса "Systema Saturnium", взятые с сайта <Ref to="https://www.lindahall.org/about/news/scientist-of-the-day/christiaan-huygens-2/">Linda Hall Library</Ref>.
```

Да, это не особо "красиво" не оставлять ссылки на **все** источники, которые подали идею или из которых был в каком-то виде использован материал. Однако такой подход позволяет не тратить кучу времени на поиск реального первоисточника (а не его "перепостов"), следить за тем, чтобы ссылка была рабочей и прочего ада, связанного с содержанием "списка источников" в актуальном состоянии.
